vedor # Guia de Início Rápido - LifeOS

## 🚀 Como Começar

### 1. **Iniciar o Projeto**

Abra o terminal na pasta do projeto e execute:

```bash
# Instalar dependências
npm install

# Iniciar o servidor de desenvolvimento
npm run dev
```

### 2. **Acessar o LifeOS**

Após iniciar o servidor, abra seu navegador e acesse:
- **URL**: `http://localhost:5173`

### 3. **Explorar as Funcionalidades**

#### **🏠 Dashboard (Painel)**
- Visualize o resumo do seu dia
- Veja progresso de hábitos e metas
- Acompanhe estatísticas diárias

#### **🔥 Hábitos**
- **Cadastrar hábitos**: Clique em "Novo Hábito"
- **Marcar conclusão**: Clique no círculo ao lado do hábito
- **Organizar por categorias**: Use as categorias para melhor organização

#### **🎯 Metas**
- **Criar metas**: Clique em "Nova Meta"
- **Definir prazos**: Estabeleça datas de início e término
- **Acompanhar progresso**: Visualize barras de progresso diárias

#### **👥 Relacionamentos**
- **Cadastrar pessoas**: Clique em "Novo Relacionamento"
- **Armazenar informações**: Nome, idade, aniversário, preferências
- **Gerenciar contatos**: Edite ou exclua quando necessário

#### **⚙️ Configurações**
- **Tema**: Alterne entre light e dark mode
- **Exportar dados**: Faça backup dos seus dados
- **Importar dados**: Restaure dados de backup
- **Limpar dados**: Reinicie o sistema (cuidado!)

## 📱 Uso no Celular

### **Abrir Barra Lateral**
- **Swipe**: Deslize da esquerda para direita na borda da tela
- **Botão**: Clique no ícone de menu (☰) no canto superior esquerdo

### **Navegação**
- **Menu lateral**: Acesse todas as áreas do LifeOS
- **Voltar**: Use o botão de fechar (X) ou clique fora da barra

## 💡 Dicas de Uso

### **Hábitos**
- **Comece devagar**: Cadastre 2-3 hábitos simples
- **Seja consistente**: Marque os hábitos diariamente
- **Use categorias**: Organize por áreas da vida (saúde, estudo, etc.)

### **Metas**
- **Seja específico**: Defina metas claras e mensuráveis
- **Defina prazos**: Estabeleça datas realistas
- **Divida em etapas**: Metas grandes em pequenas tarefas

### **Relacionamentos**
- **Mantenha atualizado**: Atualize informações regularmente
- **Use tags**: Organize por tipo de relacionamento
- **Lembre datas**: Aniversários e datas importantes

## 🔧 Configurações Importantes

### **Tema**
- **Light**: Visual limpo e branco (padrão)
- **Dark**: Visual escuro e moderno
- **Alternar**: Use o botão na barra lateral

### **Backup de Dados**
- **Exportar**: Salve seus dados em arquivo JSON
- **Importar**: Restaure dados de backup
- **Recomendado**: Faça backup semanalmente

## 🆘 Suporte

### **Problemas Comuns**
- **Página não carrega**: Verifique se o servidor está rodando
- **Dados não salvam**: Verifique o localStorage do navegador
- **Layout quebrado**: Limpe o cache do navegador

### **Resetar Dados**
1. Vá em **Configurações**
2. Clique em **Limpar Dados**
3. Confirme a exclusão
4. Recarregue a página

## 📞 Suporte Técnico

Para dúvidas ou problemas:
- Verifique o console do navegador (F12)
- Consulte este guia
- Reinicie o servidor se necessário
