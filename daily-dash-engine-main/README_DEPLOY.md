## 🌐 Deploy em 3 Passos

### **1. Vercel (Recomendado)**
```bash
# 1. Conectar ao GitHub
git init
git add .
git commit -m "Deploy LifeOS"
git remote add origin https://github.com/SEU_USUARIO/SEU_REPO.git
git push -u origin main

# 2. Conectar ao Vercel
# Acesse vercel.com > Import Project > GitHub > SEU_REPO

# 3. Deploy automático!
# URL: https://SEU-REPO.vercel.app
```

### **2. Netlify**
